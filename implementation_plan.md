# 大学城美食地图 — 网页设计方案

## 概述

为高校联盟（3-5所大学）周边学生打造的本地化美食发现平台。帮助学生快速找到附近符合口味/预算的餐厅，解决"吃什么"决策困境。

**技术栈**：纯 HTML5 + CSS3 + Vanilla JS（零依赖，轻量快速），地图使用嵌入式 iframe 或 SVG 示意图。

---

## 设计亮点

| 特性 | 说明 |
|------|------|

## 页面结构

### 1. Hero 区域
- 全屏背景（渐变 + 粒子动画）
- 主标语："发现大学城最好吃的那一家"
- 全局搜索框（支持菜名/店名/菜系模糊搜索）
- 大学快速切换 Tab（中南大学 / 湖南大学 / 湖南师范大学 / 中南林业 / 湖南农业大学）

### 2. 筛选侧边栏 / 顶栏
- 菜系分类：全部 / 🍜 中餐 / 🍣 日料 / 🌮 西餐&快餐 / 🍵 奶茶饮品 / 🥗 轻食 / 🍢 夜宵烧烤
- 价格区间：<15元 / 15-30元 / 30-50元 / 50+元
- 排序方式：综合推荐 / 评分最高 / 距离最近 / 最多人气

### 3. 双视图切换
- **卡片网格视图**：3 列响应式卡片，hover 上浮效果
- **地图视图**：SVG 示意地图 + 餐厅热力气泡标记

### 4. 餐厅卡片
- 封面图（AI 生成）
- 店名、菜系 badge、评分星级
- 人均价格、距校距离、推荐标签

### 5. 详情弹窗（Modal）
- 大图轮播
- 基本信息（地址、电话、营业时间）
- 招牌菜推荐区
- 学生评价（3条）
- 一键导航按钮

---

## 数据规模

- **5所大学**：中南大学、湖南大学、湖南师范大学、中南林业科技大学、湖南农业大学
- **25+ 餐厅**：覆盖各菜系和价格段，含真实感数据
